# Pong
## Pong — простая аркадная игра в стиле классического пинг-понга
### Автор: Анастасия Бердюгина
#### Что есть в проекте
 - Главный экран с игровой логикой (PongViewController)
 - Расширения для удобной организации кода:
   - PongViewController+BallHit.swift - реакция на столкновения мяча
   - PongViewController+EnemyPaddleBehavior.swift — поведение ракетки противника
   - PongViewController+GestureHandling.swift — управление свайпами игрока
   - PongViewController+LaunchBall.swift — запуск мяча
   - PongViewController+UIKitDynamics.swift — физика движения мяча и ракеток

#### Что происходит в игре
 - Игрок управляет своей ракеткой, чтобы отбивать мяч
 - Противник управляется компьютером 
 - Мяч движется с ускорением и отскакивает от ракеток и границ экрана
   
#### Цель: отбивать мяч как можно дольше и набирать очки
